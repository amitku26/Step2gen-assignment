Portfolio QR
Overview
A simple and elegant QR Code UI that generates a QR code for my portfolio using a public QR API. The QR can be downloaded, the URL can be copied, and a tooltip appears when hovering, following the cursor. This project uses HTML, CSS, and JavaScript, optimized with a gray–white–black theme.

Features
Generates QR code dynamically using an API ✔️ Clean UI with gray/white/black modern theme ✔️ Hover tooltip that follows cursor ✔️ Tooltip shows the portfolio URL ✔️ Click QR to open portfolio ✔️ Download QR Code as PNG ✔️ Copy portfolio URL to clipboard ✔️ Smooth hover animations
