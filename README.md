# Portfolio QR

## Overview
A lightweight and responsive web project that creates a scannable QR code for your portfolio.  
Built using **HTML, CSS, JavaScript**, and the **QRious** QR code generation library.



## Features
- Automatically generates a QR code for your portfolio URL  
- Clean and modern responsive UI  
- Download QR code as a PNG image  
- Copy portfolio URL to clipboard  
- Lightweight and fast (HTML + CSS + JS)  
- Fully customizable structure
